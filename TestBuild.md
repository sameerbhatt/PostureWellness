# Posture Wellness - Test Build

## Installation
1. Unzip PostureWellness.zip
2. Drag PostureWellness.app to /Applications folder
3. Right-click app → Open (first time only)
4. Click "Open" in security dialog

## First Run
- Grant Camera permission (required)
- Grant Notification permission (recommended)
- App runs in menu bar (no dock icon)

## Testing
- Look for 🟢/🟡/🔴 icon in menu bar
- Click icon for menu
- Try "Settings" to configure
- Try "Open Dashboard" to see analytics
- App captures posture every 60 seconds by default

## Known Issues
- macOS may show security warning (expected for test builds)
- Camera light stays on during capture (normal)
